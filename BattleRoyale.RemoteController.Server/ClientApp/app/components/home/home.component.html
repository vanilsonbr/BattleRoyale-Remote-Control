BattleRoyalle: CLI Remote Access [ALPHA]

Welcome to BattleRoyalle's Remote Terminal!

Here you can access registered client machines by using your online terminal!

By clicking in the option "Machines Connected", you will see a list of machines connected to this server at this momment

Click in the of the desired machine and a screen will appear for you to send your commands right here!

Click in the of the desired machine to check if it is connected (machines disconnected will not load their commands panel)

We developed the very first version, so keep those observations in mind:

  • We are still developing the PowerShell funcionality. Wait until the next version to use it.
  • Currently, the application receives only one command at a time. Future versions will support the and (&&) command.
  • The screens presented here are still in prototyping process. If you have any improvement suggestions, contact us, we will be ready to hear from you!
  • If you find any bug, please talk to us immediately and we will solve the problem for you!

Thank you, and enjoy the application!
